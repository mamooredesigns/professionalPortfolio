CRSN

Code the Dream's Community Ride Share Network Mobile App

My first assignment as an intern for Code the Dream was to update a mobile app that was built by a previous intern class for use with an existing web app that helps volunteer drivers coordinate with individuals who need free transportation.

Besides redesigning the user interface, I worked on the app itself, built using React Native, to update deprecated libraries, add features, and publish builds to the Apple and Google app stores.
